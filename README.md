# Irctc-Automation

**Irctc-Automation** is a fun and innovative project that utilizes Python and Selenium WebDriver to automate the Tatkal booking process on the IRCTC website. The main goal of this project is to help users make fast reservations by automating the tedious steps involved in Tatkal booking. By using Selenium, this tool simulates user interaction with the IRCTC website, speeding up the booking process.

## Features

- **Automated Tatkal Booking:** 
  - Automatically fills in passenger details and booking information.
  - Helps users complete the booking process faster by bypassing manual data entry.
  
- **Fast and Efficient:** 
  - Reduces the time required for Tatkal booking by automating the repetitive tasks.
  
- **Simple User Input:**
  - The script takes user input for journey details, such as train number, source, destination, passenger info, etc.
  
- **Error Handling:**
  - Includes basic error handling to deal with common issues such as connection problems or incorrect data.

## Tech Stack

- **Backend:** Python
- **Automation:** Selenium WebDriver
- **Web Driver:** ChromeDriver (or any other supported driver)
- **Libraries:** Selenium, WebDriver Manager
- **Web Scraping:** BeautifulSoup
